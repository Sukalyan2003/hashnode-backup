The goal is to summarize all the practice points in a blog post as follows:

First, reorganize all the points in "100 Days of machine learning" note into sections, corresponding to each stage of the process. Then, in the blog post, you first introduce the section, insert all the most important notes for that section, then explain everything you did for practice.

All these practice things should be on the same topic, so that an end to end demonstration is possible
